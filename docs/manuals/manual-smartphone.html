Manual de Teste no Smartphone - NEXORA (Para Leigo)

1. Objetivo

Este guia mostra como testar o NEXORA pelo celular sem conhecimento técnico.

2. O que voce precisa

  • Celular com internet
  • Servidor NEXORA ligado
  • Link de acesso remoto ativo

3. Teste rapido pelo navegador

  1. Abra o navegador do celular.
  2. Acesse o link gerado no painel admin (Host Externo).
  3. Faça login com usuario de teste.
  4. Abra uma solicitacao e confira notificacoes.

4. Teste pelo APK (quando disponivel)

  1. Baixe o APK no painel admin.
  2. Instale no celular.
  3. Abra o app: ele conecta automaticamente usando bootstrap.
  4. Valide abertura de chamados e leitura de alertas.

5. O que validar

  • Login funcionando.
  • Abertura de solicitacao.
  • Atualizacao de status.
  • Notificacoes recebidas.
  • Acesso ao infografico e area de usuario.

6. Se der erro

  • Atualize a pagina.
  • Peça novo usuario de teste no painel admin.
  • Verifique se host externo esta ativo.
