Manual de Uso do Usuario - NEXORA v24.8

1. Para que serve

Este manual mostra como qualquer usuario pode usar o sistema para abrir solicitacoes, acompanhar status e receber notificacoes.

2. Como entrar

  1. Acesse http://127.0.0.1:8080.
  2. Digite seu e-mail e senha.
  3. Se tiver erro, contate o administrador.

3. Abrindo uma solicitacao

  1. Entre na area de servicos.
  2. Escolha o modulo (exemplo: Workflow ou Billing).
  3. Descreva o pedido de forma clara.
  4. Defina prioridade.
  5. Envie e guarde o numero da solicitacao.

4. Acompanhando progresso

Cada solicitacao passa por estados: open, in_progress, resolved, closed ou escalated.

5. Notificacoes

  • Use a lista de notificacoes para ver alertas.
  • Clique em confirmar leitura (ACK) para limpar pendencias.
  • No smartwatch, use ACK, Escalar ou Soneca 10m.

6. Mobile e smartwatch

O layout se adapta automaticamente para celular e relogio inteligente, com foco em acoes rapidas e alertas criticos.

7. Dicas para bom uso

  • Escreva titulos objetivos nas solicitacoes.
  • Inclua dados importantes no texto.
  • Acompanhe prazos e notificacoes diariamente.
  • Evite criar pedidos duplicados.

8. Suporte

Se algo nao funcionar, informe o codigo da solicitacao e horario do problema para o admin.
