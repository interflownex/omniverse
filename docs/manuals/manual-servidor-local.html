Manual de Uso - Servidor Local NEXORA v24.8

1. O que e este manual

Este guia ensina, passo a passo e sem termos tecnicos complexos, como iniciar, testar e desligar o servidor local do NEXORA.

2. Requisitos basicos

  • Windows com PowerShell
  • Acesso a internet para instalar dependencias
  • Pasta do projeto: C:\interflownex\omniverse

3. Primeira preparacao

  1. Abra PowerShell na pasta do projeto.
  2. Execute .\scripts\bootstrap.ps1.
  3. Espere o fim da instalacao automatica.
Se aparecer aviso de permissao, execute PowerShell como administrador.

4. Como ligar o servidor

  1. Execute .\scripts\up.ps1.
  2. Acesse http://127.0.0.1:8080.
  3. Verifique saude com .\scripts\healthcheck.ps1.

5. Como validar internet da maquina

Execute .\scripts\validate-host-internet.ps1 antes e depois da configuracao remota. O resultado deve mostrar conexoes OK.

6. Como desligar com seguranca

  1. Para parar: .\scripts\down.ps1
  2. Para resetar ambiente: .\scripts\reset.ps1
  3. Para remover tudo local: .\scripts\uninstall.ps1

7. Teste remoto por link

  1. Execute .\scripts\remote-test-pack.ps1.
  2. Compartilhe o link gerado com o testador.
  3. Ao final, execute .\scripts\revoke-remote-tester.ps1 e .\scripts\unpublish-remote.ps1.

8. Solucao de problemas

  • Se nao abrir o sistema: rode novamente o healthcheck.
  • Se login falhar: confirme credenciais padrao no dashboard.
  • Se memoria exceder: rode .\scripts\check-memory-budget.ps1.
